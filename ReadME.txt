Первой задачей стало анализ имеющихся данных. Для открытия самих данных мы воспользовались Pyspark'ом. 
Программа работае следующим образом. Мы береме изначальные данные и с помощью методов Pyspark открываем данные,
которые посылат к нам. Затем идет дальнейшая работа над каждый из аспектов, то есть происходит агреграция 
каждого из необходимых компонент относительно дней и переводим данные в мегабайты для дальнейшей работы. 
Дальше я агреггирую дату в одно. То есть на вход нам посылаются данные общие а на выход нам будет поступать 
обработанные агрегированные по дням данные пользователей.
